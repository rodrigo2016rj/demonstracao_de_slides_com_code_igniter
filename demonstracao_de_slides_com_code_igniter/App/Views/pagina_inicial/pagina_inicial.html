Página Inicial

Selecione qual slide deseja ver o resultado pelo menu do cabeçalho do sistema. Este sistema demonstra o resultado de três formas de fazer slide:

No primeiro link, JavaScript, o sistema mostra o resultado do slide feito apenas com JavaScript, sem o uso de quaisquer frameworks para JavaScript. Por isso chamei de JavaScript puro.

No segundo link, JQuery, o sistema mostra o resultado do slide feito com JQuery.

No terceiro link, React, o sistema mostra o resultado do slide feito com React.
